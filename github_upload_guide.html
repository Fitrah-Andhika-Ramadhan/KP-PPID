Upload Your PPID Dispora Jabar Portfolio to GitHub

Follow these steps to upload your portfolio project to GitHub. This guide will help you create a new repository and push your code to it.

Step 1: Create a GitHub Account (if you don't have one)

If you don't already have a GitHub account, create one at:

Create GitHub Account

Step 2: Create a New Repository on GitHub

Once you're logged in to GitHub:

  1. Click the "+" icon in the top-right corner and select "New repository"
  2. Name your repository (e.g., "ppid-portfolio")
  3. Add a description (optional): "PPID Dispora Jabar Portfolio Website"
  4. Keep it as a Public repository
  5. Do NOT initialize with a README, .gitignore, or license
  6. Click "Create repository"
Create New Repository

Step 3: Open Command Prompt

Open Command Prompt on your computer and navigate to your project directory:

cd D:\ppidporto\Portfolio

Step 4: Initialize Git Repository

Initialize a new Git repository in your project folder:

git init

Step 5: Create .gitignore File

Create a .gitignore file to exclude unnecessary files from your repository:

copy nul .gitignore

Then open the .gitignore file in a text editor and add the following content:

/vendor/
/node_modules/
/.env
/.env.backup
/.phpunit.result.cache
/Homestead.json
/Homestead.yaml
/npm-debug.log
/yarn-error.log
/.idea
/.vscode
/public/storage
/storage/*.key

Step 6: Add Your Files to Git

Add all your project files to the Git repository:

git add .

Step 7: Commit Your Changes

Commit your files with a descriptive message:

git commit -m "Initial commit of PPID Dispora Jabar Portfolio"

Step 8: Add Remote Repository

Link your local repository to your GitHub repository. Replace YOUR_USERNAME with your GitHub username and REPOSITORY_NAME with the name you gave your repository:

git remote add origin https://github.com/YOUR_USERNAME/REPOSITORY_NAME.git

Step 9: Push to GitHub

Push your code to GitHub:

git push -u origin master

You may be prompted to enter your GitHub username and password.

Note on GitHub Authentication:

GitHub no longer accepts password authentication from the command line. You'll need to use a personal access token instead of your password.

  1. Go to GitHub Personal Access Tokens
  2. Click "Generate new token"
  3. Give it a name (e.g., "Portfolio Upload")
  4. Select the "repo" scope
  5. Click "Generate token"
  6. Copy the token (you won't be able to see it again)
  7. Use this token instead of your password when prompted

Step 10: Verify Your Upload

Go to your GitHub repository page to verify that your files have been uploaded successfully:

https://github.com/YOUR_USERNAME/REPOSITORY_NAME

Next Steps

After uploading your project to GitHub, you can:

  • Enable GitHub Pages to host your static website
  • Share your repository link with others
  • Continue to update your code and push changes to GitHub

Need Help?

If you encounter any issues during the upload process, refer to the GitHub documentation or contact GitHub support.
