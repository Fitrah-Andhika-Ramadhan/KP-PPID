GitHub Repository Setup

To upload your PPID Dispora Jabar Portfolio to GitHub, you need to create a new repository and connect it to your local project.

Create New GitHub Repository

Important: When creating your repository on GitHub:

  • Use the repository name you entered above
  • Make it Public
  • Do NOT initialize with README, .gitignore, or license

After Creating Your Repository:

1. Open Command Prompt and run these commands (replace USERNAME and REPO-NAME with your values):

cd D:\ppidporto\Portfolio
git remote add origin https://github.com/USERNAME/REPO-NAME.git
git push -u origin main

2. When prompted for credentials:

  • Enter your GitHub username
  • For password, use a Personal Access Token (not your GitHub password)

3. Verify your repository was uploaded by visiting:

https://github.com/USERNAME/REPO-NAME
